# 🏠 Rentify - Online Rental Property Platform

Rentify is a full-stack web application inspired by Airbnb. It allows users to browse rental properties, and hosts to list their properties for rent. The platform is built using **React** for the frontend and **Spring Boot** for the backend, with **MySQL** as the database.

## 📌 Project Features

### 👤 Users
- Register/Login
- View all available rental properties
- Book a property
